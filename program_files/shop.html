    What are themes?

    nuken Themes change the appearance and layout of your nuken workspace. Make sure you have v5.0 or higher before downloading a theme! To use nuken Themes, click on a thumbnail image in the themes library - this will make a popup menu appear, asking you to confirm your decision. From there, click the "Download theme" button to download your file. A .zip file will be downloaded to your device's default downloads location. From there, extract the contents of the .zip file and extract it to this file path:

    nuken/program_files/themes/selected/

    Then start the nuken offline web app. Your selected theme wil automatically load in. Enjoy! You can view more instructions in the nuken Handbook add-on.

    NOTE: Students using nuken Education cannot download themes.

    Okay, I got it!

    What are add-ons?

    nuken Add-ons bring additional functionality to your nuken workspace. Make sure you have v5.0 or higher before downloading an add-on package! To use nuken Add-ons, click on a thumbnail image in the add-ons library - this will make a popup menu appear, asking you to confirm your decision. From there, click the "Download add-on" button to download your file. A .zip file will be downloaded to your device's default downloads location. From there, extract the contents of the .zip file and extract it to this file path:

    nuken/program_files/addons/active/

    You'll notice three slot folders in this file location. This is what order they will appear in when you view them in the offline nuken web app, so drop the files into whatever slot you want. Then start the nuken offline web app. Your active add-ons will be made available. Enjoy! You can view more instructions in the nuken Handbook add-on. (NOTE: Students using nuken Education cannot download add-ons from the nuken Shop.)

    What are packages?

    nuken Packages come pre-installed with things like exclusive themes and add-ons, added functionality, or a modified user interface. There's no folder to drop these into - simply download and extract these wherever you want! nuken Packages are pretty much special custom versions of nuken. Enjoy!

    NOTE: Students using nuken Education cannot download packages directly.

    Okay, I got it!

    What are templates?

    nuken Templates allow you to import custom projects and code into your nuken workspace. You can access your installed templates in the import menu. You can only have one nuken Template pack active at a time - just extract the template pack file to:

    nuken/program_files/templates/loaded/

    NOTE: Students using nuken Education cannot download templates.
