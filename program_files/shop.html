What are themes?

nuken Themes change the appearance and layout of your nuken workspace. Make sure you have v4.0 or higher before downloading a theme! To use nuken Themes, download the 'theme.zip' file and extract it to this file path:

nuken/program_files/themes/selected/

Then start the nuken offline web app. Your selected theme wil automatically load in. Enjoy!

What are add-ons?

nuken Add-ons bring additional functionality to your nuken workspace. Make sure you have v4.0 or higher before downloading an add-on package! To use nuken Add-ons, download the 'add-on.zip' file and extract it to this file path:

nuken/program_files/addons/active/

Then start the nuken offline web app. Your active add-ons will be available onscreen. Enjoy!
